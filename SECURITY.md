# Security Policy

Prompt Hub is a local-first browser application. It does not send workspace data to a server.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

If you find a security issue, contact the maintainer privately at:

```txt
andriipap01@gmail.com
```

Include:

- a short description of the issue
- reproduction steps
- affected browser/version if relevant
- potential impact

## Supported Versions

Security fixes target the latest version on `main`.
